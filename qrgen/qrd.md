# WiFi QR 코드 생성기 명세서

## 1. 개요
Next.js와 Tailwind CSS로 구축된 웹 애플리케이션으로, 사용자가 자신의 WiFi 네트워크용 QR 코드를 생성할 수 있도록 합니다. 스마트폰 카메라로 생성된 QR 코드를 스캔하면 지정된 WiFi 네트워크에 연결할지 묻는 메시지가 기기에 자동으로 표시됩니다.

## 2. 기술 스택
- **프레임워크:** Next.js (최신 React를 위한 App Router 권장)
- **스타일링:** Tailwind CSS (반응형, 모던 UI)
- **QR 코드 라이브러리:** `qrcode.react` (QR 코드를 SVG/Canvas로 쉽게 렌더링하기 위함)
- **아이콘 (선택사항):** `lucide-react` (다운로드, 와이파이, 자물쇠 같은 UI 아이콘용)

## 3. 주요 기능
- **사용자 입력:**
  - **네트워크 이름 (SSID):** 텍스트 입력 (필수). WiFi 네트워크의 이름.
  - **비밀번호:** 텍스트/비밀번호 입력 창. 비밀번호를 보이거나 숨길 수 있는 토글 기능 포함.
  - **암호화 방식:** 드롭다운 또는 라디오 버튼 (WPA/WPA2/WPA3, WEP, 없음). 기본값: WPA/WPA2/WPA3.
  - **숨겨진 네트워크 여부:** SSID가 숨겨져 있는지를 나타내는 체크박스. 기본값: false (숨기지 않음).
- **QR 코드 생성:**
  - 동적 생성: 사용자가 입력할 때마다 실시간으로 QR 코드가 업데이트되거나 "생성" 버튼 클릭 시 생성.
  - 표준 포맷: `WIFI:T:{암호화방식};S:{ssid};P:{비밀번호};H:{숨김여부};;`
- **추가 액션:**
  - **다운로드 기능:** 생성된 QR 코드를 인쇄하거나 공유하기 위해 PNG 또는 SVG 이미지로 다운로드 하는 기능.

## 4. UI/UX 디자인
- **테마:** 깔끔하고 현대적인 구조이며 모바일 화면을 우선적으로 고려한(mobile-first) 디자인.
- **레이아웃:** 좌측/상단에는 폼 영역을, 우측/하단에는 생성된 QR 코드를 미리볼 수 있게 배치하는 중앙 정렬 형태의 카드 레이아웃.
- **상호작용:** 버튼에 부드러운 호버(hover) 효과 제공, 폼 검증(예: SSID 필수 입력 처리) 안내, 에러 발생 시 명확하게 인지할 수 있는 상태 표시 등.

## 5. 구현 단계
1. **프로젝트 설정:**
   - `qrgen` 디렉터리에 Tailwind CSS가 포함된 Next.js 프로젝트를 초기화합니다.
   - 불필요한 기본 보일러플레이트 코드를 정리합니다.
2. **의존성(Dependencies) 라이브러리 설정:**
   - `qrcode.react` 라이브러리 (또는 유사한 패키지)를 설치합니다.
3. **컴포넌트 개발:**
   - `WiFiForm`: 사용자 입력을 받기 위한 컴포넌트.
   - `QRCodeDisplay`: 입력된 문자열을 기반으로 QR 코드를 화면에 그려주는 컴포넌트.
4. **통합 로직 구현:**
   - 메인 페이지 또는 부모 컴포넌트에서 입력 상태(SSID, 암호, 암호화 방식, 숨김 여부)를 관리합니다.
   - 입력받은 상태를 `WIFI:...` 문자열 형식에 맞춰 포맷팅합니다.
5. **다운로드 기능:**
   - 화면에 렌더링된 Canvas를 캡처하거나 SVG를 다운로드하는 메커니즘을 구현합니다.
6. **테스트:**
   - 생성된 QR 코드를 iOS 및 안드로이드 기본 카메라 앱으로 직접 스캔해 보아 호환성을 확인합니다.
