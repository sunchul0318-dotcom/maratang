1. 프로젝트 개요
프로젝트 명: 메이크오버 플레이어블 광고 클론 (Grooming & Clean-up)

목표: 사용자가 도구를 선택하고 화면을 '문질러서' 지저분한 요소를 제거하는 인터랙션 구현.

핵심 메커니즘: 마스킹 레이어 기술을 활용한 이미지 상태 변화.

2. 게임 핵심 루프 (Core Loop)

문제 발생(Problem): 사물의 ( 자동차, 탱크, 비행기 ) 지저분한 상태를 노출하여 시각적 불편함 유도. ( assets/model 폴더 참고 )

도구 선택(Tool Selection): 우측 사이드바에서 적절한 도구를 탭하여 활성화. ( assets/tools/ 폴더 참고 )

정화 인터랙션(Cleaning): 드래그/클릭을 통해 지저분한 레이어를 지워나감.

보상 및 전환(CTA): 모든 과정 완료 후 깨끗해진 결과물 노출 및 앱스토어 이동 버튼(Call to Action) 등장.

3. 주요 기능 요구사항

도구 시스템: 각 도구(가위, 면도기, 드라이버, 크림 등)는 특정 타겟 레이어에만 반응하도록 설정 가능해야 함.

레이어 마스킹: 유저의 터치 좌표를 따라 '오염 레이어'의 알파(Alpha) 값을 0으로 만드는 로직.

단계별 트리거: 오염도가 특정 수치(예: 90%) 이하로 떨어지면 자동으로 다음 단계나 연출로 전환.

비주얼 피드백: 문지를 때 발생하는 파티클(털 뭉치, 거품 등) 및 캐릭터의 리액션 애니메이션.

4. 기술적 구현 제언

Phaser 를 이용하여 제작합니다. 만일 물리가 필요하다면 matter.js 를 활용합니다.

더 필요한 이미지가 있다면 일단 더미 이미지를 만들어서 진행해주세요.

게임의 핵심 플로우를 다시 설명합니다.
처음에 게임을 시작하면, 탱크, 자동차, 비행기중하나를 고릅니다.
하나를 고르면 더러워진 사물의 모습이 보입니다.
우측에서 적당한 도구를 고릅니다.
이제 지저분한 부위를 드래그 하면 해당 좌표를 주변으로 알파값을 점점 줄여나가서 결국 깨끗한 이미지만 노출되게 됩니다.
만일 도구가 알맞지 않다면 드래그 해도 아무런 일도 일어나지 않습니다.
승리 조건은 일정 퍼센트 이상 깨끗하게 한 경우입니다.
